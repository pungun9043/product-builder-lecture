# Lottery Number Generator

## Overview

This is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, and it's designed to be a fun and easy-to-use tool for anyone who wants to try their luck at the lottery.

## Design and Features

*   **Modern Design:** The application has a clean and modern design, with a focus on usability and aesthetics.
*   **Responsive Layout:** The layout is fully responsive, so it looks great on any device, from a desktop computer to a mobile phone.
*   **Interactive Elements:** The application includes interactive elements, such as a button that glows when you hover over it, to make the user experience more engaging.
*   **Random Number Generation:** The core feature of the application is its ability to generate random lottery numbers. It generates 6 unique numbers between 1 and 45.
